# Image Compression using K-Means Clustering

This project demonstrates how **K-Means Clustering** can be used for **image compression** by reducing the number of colors in an image.

Instead of storing millions of colors, the algorithm groups similar colors into clusters and replaces them with cluster centroids.

## 📌 Features
- Image compression using K-Means
- Adjustable number of clusters (K)
- Visualization of original vs compressed image
- Simple Python implementation

## ⚙️ Technologies Used
- Python
- NumPy
- Scikit-learn
- Matplotlib
- OpenCV
